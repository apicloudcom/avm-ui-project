# ProgressBar 进度条

<code src="./demos/index.tsx"></code>

### 属性

| 参数    | 说明   | 类型     | 默认值 |
| ------- | ------ | -------- | ------ |
| percent | 百分比 | `number` | `0`    |

### CSS 变量

| 属性          | 说明       | 默认值    | 全局变量                         |
| ------------- | ---------- | --------- | -------------------------------- |
| trackWidth   | 线条宽度   | `3px`     | `--adm-progress-bar-track-width` |
| trackColor  | 线条的颜色 | `#1677FF` | `--adm-progress-bar-track-color`  |
| fillColor  | 填充的颜色 | `#E5E5E5` | `--adm-progress-bar-fill-color`  |
