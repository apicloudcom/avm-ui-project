### 属性

| 属性    | 说明                                                                                            | 类型                                  | 默认值    |
| ------- | ----------------------------------------------------------------------------------------------- | ------------------------------------- | --------- |
| content | 徽标内容：如果传 `null` `undefined` `''` 或不传，则不显示徽标 | `React.ReactNode` | -         |
| color   | 徽标背景色                                                                                      | `string`                              | `#FF411C` |
| isDot   | 是否显示为小红点，如果传 true，则显示小红点，设置content则无效                                                       | `Boolean`            | `false` |
| right | 相对于最右边，向左的偏移量 | `-` |  `0`    |
| top   | 相对于最上边，向下的偏移量 |  `-` | `0`    |

## 注意：该组件中传参暂不支持标签属性
